Component Library

The UI is built from a set of reusable, single-purpose React components. This approach promotes consistency, maintainability, and faster development.

Core Layout Components

  • App.tsx: The root component that assembles the main layout, including the sidebars, header, message list, and input area. It also handles global keyboard shortcuts.
  • Header.tsx: A smart component that conditionally renders one of two versions: a `DashboardHeader` when no chat is active, or a `ConversationHeader` with chat-specific controls.
  • ConversationList.tsx: The collapsible left sidebar that lists all conversations and includes the search functionality.
  • BookmarkedMessagesPanel.tsx: The collapsible right sidebar for displaying bookmarked messages.
  • StatusBar.tsx: The persistent footer that displays conversation and global usage metrics.

Chat & Message Components

  • MessageList.tsx: Renders the list of messages for the active conversation by mapping over the data and using the `MessageBubble` component for each one.
  • MessageBubble.tsx: A highly complex component responsible for rendering a single message. It handles different senders (user, agent, system), markdown parsing, syntax highlighting, the message action toolbar, long message collapsing, and displaying metadata.
  • MessageInput.tsx: The user input area at the bottom, including the text area, send button, and file attachment logic.
  • ManualSuggestions.tsx: Renders the buttons for manually selecting an agent when in 'Manual' mode.
  • PlanDisplay.tsx: A specialized component rendered inside a `MessageBubble` to display a multi-step plan from the AI Manager.

Modal Components

All modals share a consistent look and feel and are managed by the useModalManager hook.

  • SettingsModal.tsx: The main application settings panel for configuring global API keys, agent profiles, and preferences.
  • ConversationSettingsModal.tsx: The settings panel for the currently active conversation, including system instruction overrides and feature flags.
  • HistoryModal.tsx: Displays the AI-generated summary and topic list for a conversation.
  • CognitiveInspectorModal.tsx: The "Workflow Visualizer" that displays the AI's step-by-step process as an interactive timeline.
  • PromptInspectorModal.tsx: A developer-focused modal to view the exact raw prompt and response for a given message.
  • TeamGeneratorModal.tsx: A multi-step wizard for generating a new team of AI agents from a high-level goal.
  • ...and several others for specific actions like API usage, agent stats, and HTML previews.
